# 79NG Fjord Model
Code for a realistic numerical ocean model of the 79째 North Glacier (79NG) fjord in Northeast Greenland, using the General Estuarine Transport Model (GETM, <https://getm.eu/>).

The model is based on the idealized setup by Reinert et al. (2023),
using files from Reinert (2023).
It is currently in an early development phase.
Feel free to reach out if you have any questions or would like to collaborate.

Developed by Markus Reinert ([ORCID: 0000-0002-3761-8029](https://orcid.org/0000-0002-3761-8029)) at the Leibniz Institute for Baltic Sea Research Warnem체nde (IOW).
